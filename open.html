⚠️ Önemli Uyarı

Bu dosyayı çift tıklayarak açtınız. YouTube API'sinin çalışması için projeyi bir HTTP sunucusu üzerinden açmanız gerekiyor.

KOLAY VS Code ile

  1. VS Code'u açın
  2. "Live Server" extension'ını kurun
  3. index.html'e sağ tıklayın
  4. "Open with Live Server" seçin

Python ile

Terminal'de proje klasöründe:

python -m http.server 8080

Sonra tarayıcıda: http://localhost:8080

PHP ile

php -S localhost:8080

Sonra tarayıcıda: http://localhost:8080

Node.js ile (npx)

npx http-server -p 8080 -o

Otomatik olarak tarayıcıda açılır.
